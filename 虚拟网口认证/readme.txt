为您带来自由网路

使用说明:

0.本软件的制作是为解决在多线负载均衡状态下电信校园网(Protal协议,PPPoE请使用华中科技大学的mentoHUST自行魔改尝试)的一机多认证问题,为虚拟wan口认证电信用户,实现带宽叠加
  (通俗一点,六个50m加起来变300m)

1.运行CUIThacker.exe,并请根据提示填写您要接入的宽带账号数量,账号名,密码和网口IP
	网口ip的获取方法:

		如果你使用路由器:
        			1.使用ssh或者telnet连接到你刷好的路由器(详细见https://feverking.cn/cuit)
        			2.输入命令:ifconfig
        			3.找到若干个开头相同的ip(具体数目取决于你插了几根网线)
        			4.将其输入config.ini

		如果你使用电脑:
			1.打开cmd,运行ipconfig,确保网线已插入
			2.一般找到以太网适配器的IP,输入即可
